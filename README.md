# 🎵 AI Song Style Prompts

A curated collection of high-quality, detailed style prompts for AI music generators like Suno, Udio, and others.

Each style is broken down with production notes, vocal characteristics, structure, and ready-to-copy prompts.

## Styles Included
- [Queen - Another One Bites the Dust](./styles/queen-another-one-bites-the-dust.md)

## How to Use
1. Browse the `styles/` folder
2. Copy the **Ready-to-use Prompt**
3. Paste into your favorite AI music tool
4. Customize the `[TOPIC]` or lyrics

## Adding New Styles
Use the [TEMPLATE.md](./TEMPLATE.md) to create consistent new entries.
